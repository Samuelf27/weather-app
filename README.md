## 📌 Sobre o projeto

Aplicação que consome a **[Open-Meteo API](https://open-meteo.com/)** (gratuita e sem chave) para mostrar o clima atual e a previsão dos próximos dias de qualquer cidade do mundo. Detecta automaticamente a sua localização e adapta o fundo conforme a temperatura.

## ✨ Funcionalidades

- 📍 **Geolocalização** automática ao abrir (com fallback para São Paulo)
- 🔎 Busca por **qualquer cidade** (geocoding em português)
- 🌡️ Clima atual: temperatura, sensação, umidade e vento
- 📅 **Previsão de 5 dias** com máximas e mínimas
- 🎨 Fundo dinâmico que muda com a temperatura
- 🌧️ Ícones e descrições traduzidos a partir dos códigos WMO

## 🛠️ Tecnologias

HTML5 · CSS3 (glassmorphism) · JavaScript (Fetch API, async/await) · Open-Meteo API

## 🚀 Como rodar

```bash
git clone https://github.com/Samuelf27/weather-app.git
cd weather-app
# abra index.html no navegador (ou use Live Server)
```

> 💡 A API Open-Meteo é gratuita e **não exige chave de API**.
